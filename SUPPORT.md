# Support

Wavee support happens through GitHub.

## Questions

Use GitHub Discussions if they are enabled. If not, open an issue and choose the closest template.

## Bugs

Open a bug report with:

- Operating system and version
- Wavee version or commit
- Steps to reproduce
- Expected behavior
- Actual behavior
- Logs or screenshots when helpful

Do not include secrets, license keys, private files, or personal audio.

## Security

For suspected vulnerabilities, follow [SECURITY.md](SECURITY.md). Do not open a public issue for security reports.
